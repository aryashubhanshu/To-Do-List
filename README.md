# To-Do-List

A To-Do List in which we can easily create and remove the task. 
<br>
We can also categorize different lists by simply tapping into /{nameOfList}.

<br>
Used Tech Stacks: HTML, CSS, JavaScript, NodeJS, Express, Mongoose, EJS.
